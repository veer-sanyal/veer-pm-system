# /debrief $ARGUMENTS

Run right after a networking, externship, or client call, while details are fresh. Read the contact's `alumni-tracker.md` row and the prep sheet in `networking/` first.

1. Capture in Veer's words: how it went, what was learned, what was promised (by either side), and any names offered — each name is a warm-intro shortlist candidate.
2. Persist: update the `alumni-tracker.md` row, the external-relationships log and P3 ledger (funnel counts) in `ledgers.md`, and mark the prep sheet historical.
3. **Same-day thank-you.** The last step is where loops die (pattern P4). Draft it now, fold in any ask that was missed live (the referral ask, the who-else-should-I-talk-to), and get it sent before the session ends. The research window is 24-48h; same-day wins when the draft is already open.
4. Name the single next physical action and open it as a tripwire with a slug (memory.md + `tripwires_opened` in the telemetry line).
5. Commit, push, telemetry line (type `debrief`).
